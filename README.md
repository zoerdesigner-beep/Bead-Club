# 拼豆图纸生成器 V1

一个零依赖静态网站，用于把图片转换成基于 MARD 291 色板的拼豆图纸，并导出 PNG。

## 功能

- 上传 JPG、PNG、WebP 图片。
- 设置图纸宽度和最大颜色数。
- 保留透明背景或填充白色背景。
- 将颜色映射到 `mard-291.json` 中的 MARD 291 色号。
- 预览网格图纸、查看用料统计。
- 下载包含图纸和图例的 PNG。

## 本地运行

由于浏览器直接打开本地 HTML 时可能无法读取 `mard-291.json`，建议使用本地静态服务器：

```bash
python3 -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

## 说明

MARD 291 色值为社区整理参考，屏幕显示和实物豆子可能存在色差。请仅上传你拥有使用权的图片。
